Raspibat

Sensor DHT22 · Temperatura y humedad en tiempo real

RaspiBat es un proyecto de IoT basado en Raspberry Pi que monitoriza temperatura y humedad con un sensor DHT22 y envía los datos cada 10 minutos a Firebase.

Incluye una cámara que puede capturar fotos bajo demanda desde esta web o de forma automática al detectar movimiento con un sensor PIR.

Las imágenes se suben a Cloudinary y se muestran aquí en la galería, junto con el gráfico en tiempo real y el listado de las últimas lecturas del sensor.

La ubicación del dispositivo se visualiza en el mapa, y el acceso a la captura en vivo está protegido con inicio de sesión.

Todo el sistema está desplegado en la nube (Firebase Hosting y Realtime Database) para consultar los datos desde cualquier dispositivo.
